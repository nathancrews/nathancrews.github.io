Point Cloud Windows Shell Extension

Quickview point cloud .LAS/.LAZ files in Windows File Explorer. Right click on .LAS/.LAZ files to produce thumbnail images.
Video
Project Details
Project Details
LAS/LAZ Pointcloud Windows Shell Extension: This "quick viewer" shell extension provides a Windows File Explorer right click menu option to quickly generate 3D preview images for large and small .las and .laz files. Typical processing time for 100mb .las 25mb .laz files is about 2 seconds. (Disclaimer: Processing time depends on compute hardware)
Files are processed in parallel using available CPU cores. A second option will create a merged 3D image for all selected .las/laz files. Both menu options support multi-file select and directory processing. Pre-built Windows x64 installer available here
Languages: C++, C#
Build Tools: Visual Studio 2022
Platforms: Windows
Framework: Win API/ATL/Shell, .net
Libraries: LASLib, PDAL, Open3D, Intel TBB
